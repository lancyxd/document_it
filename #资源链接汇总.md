  **一 版本安装和升级**

  安装 go get github.com/astaxie/beego

  获取最新版本的 bee 工具 go get -u github.com/beego/bee

  更新 beego 框架 go get -u github.com/astaxie/beego

   

  **二 bee工具安装及使用**

  bee路径 $GOPATH/bin

  bee

  bee -h 查看帮助信息

  bee version 查看版本

  bee new myproject(创建web项目,须在$GOPATH/src目录下执行)

  bee api apiproject//用来创建 API 应用的

  bee run $GOPATH/src/appname 

   

  **三 快速入门( cd $GOPATH/src/quickstart )**

  bee new quickstart

  bee run

  访问:http://localhost:8080

  1)路由设置：beego.Router("/user", &controllers.UserController{})

  2)Controller：Go 的嵌入方式，也就是 MainController 自动拥有了所有 beego.Controller 的方法。

  type UserController struct {

  beego.Controller

  }

   

  func (c *UserController) Get() {

  c.Ctx.WriteString("hello world")

  }

  3)model逻辑：数据库等操作，是逐步抽象的过程，一般我们会在 Model 里面处理一些数据读取

  4)view编写

  **MVC模式**

  Model负责的是数据，这里的数据不仅仅限于数据本身，还包括处理数据的逻辑

  View负责数据的表现形式，将数据和数据的变化呈现给用户

  Controller负责用户的输入，将用户的命令转化成消息传递给model或view，是一个翻译者

  **MV观察者模式**(V观察者，M被观察者)
